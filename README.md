# 休憩時間表示ページ

現在時刻を基準とした1時間の休憩時間を表示するシンプルなウェブページです。

## 🌐 Live Demo

**[https://hiroyukiisoe.github.io/breaktime/](https://hiroyukiisoe.github.io/breaktime/)**

## 📝 概要

ページにアクセスすると、その瞬間の時刻から1時間後までの休憩時間が「休憩入ります。（15:05〜16:05）」の形式で表示されます。表示されたテキストはワンクリックでクリップボードにコピーできます。

## ✨ 機能

- **自動時刻生成**: アクセス時の現在時刻から1時間の休憩時間を自動計算
- **24時間表記**: HH:MI形式でゼロパディング対応
- **日付跨ぎ対応**: 23:30アクセス時は00:30まで表示
- **ワンクリックコピー**: 表示テキストをクリップボードにコピー
- **視覚的フィードバック**: コピー成功時のメッセージ表示
- **レスポンシブデザイン**: モバイル・デスクトップ対応

## 🛠️ 技術仕様

- **HTML5** + **CSS3** + **JavaScript (ES6+)**
- **単一ファイル構成**: `index.html`のみ
- **Clipboard API**: モダンブラウザのコピー機能
- **GitHub Pages**: 静的サイトホスティング

## 🚀 使用方法

1. [GitHub Pages リンク](https://hiroyukiisoe.github.io/breaktime/)にアクセス
2. 表示された休憩時間を確認
3. 必要に応じて「コピー」ボタンでテキストをコピー
4. チャットやメッセージアプリに貼り付けて使用
