### 概要

GitHub Pages にアクセスした際に、現在時刻を基準とした 1 時間の休憩時間を表示する静的なウェブページを作成します。ページの表示テキストは「休憩入ります。（HH24:MI〜HH24:MI）」という形式です。

### 要件定義

#### 1. 表示内容

- ページにアクセスすると、以下の形式のテキストが画面に表示されること。
  ```
  休憩入ります。（HH24:MI〜HH24:MI）
  ```
- テキストはページの中央に表示されることが望ましい。

#### 2. 時間の動的生成

- **開始時刻**: ページにアクセスした時点の現在時刻を「時:分」の 24 時間表記（`HH:MI`）で表示します。
  - 例: 午後 3 時 5 分の場合 → `15:05`
- **終了時刻**: 開始時刻の 1 時間後の時刻を「時:分」の 24 時間表記（`HH:MI`）で表示します。
  - 例: 開始時刻が `15:05` の場合 → `16:05`
  - 例: 開始時刻が `23:30` の場合 → `00:30`
- 時・分が 1 桁の場合は、先頭に`0`を付与して常に 2 桁で表示してください（ゼロパディング）。

#### 3. 実装方法

- **言語**: HTML, CSS, JavaScript を使用します。
- **ファイル構成**: `index.html`という単一のファイルに、HTML, CSS, JavaScript のコードをすべて含めてください。
- **処理の流れ**:
  1.  ページの読み込みが完了したタイミングで JavaScript を実行します。
  2.  JavaScript で現在時刻を取得します。
  3.  取得した時刻と、その 1 時間後の時刻を計算し、指定のフォーマット（`HH:MI`）に整形します。
  4.  整形した時刻を HTML の特定の要素に埋め込み、ページに表示します。

#### 4. コピー機能

- 表示されている休憩時間のテキストをクリップボードにコピーするボタンを追加します。
- ボタンは「コピー」という文字で表示し、休憩時間テキストの下に配置します。
- ボタンをクリックすると、「休憩入ります。（HH:MI〜HH:MI）」の形式のテキストがクリップボードにコピーされること。
- コピー成功時には一時的に「コピーしました」等のフィードバックを表示することが望ましい。
